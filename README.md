# 重启今天

《重启今天》是一款网页 2D 时间管理 + 轻解谜 Demo。玩家扮演一名学生，在重要提交日前反复经历同一天，通过安排学习、休息、社交、兴趣和求助，在失败重启中保留线索，找到既能完成目标又不失去自我的一天。

## 在线/本地运行

这是纯前端项目，不需要安装依赖。

直接用浏览器打开 `index.html` 即可体验。也可以在项目目录启动静态服务器：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000/index.html
```

## 文件说明

- `index.html`：游戏页面结构
- `style.css`：界面样式
- `script.js`：游戏逻辑
- `重启今天-游戏设计赛道参赛方案.md`：参赛方案
- `游戏说明手册.md`：玩家说明
- `成就系统设计.md`：成就与反馈系统设计

## 玩法核心

- 一天分为 6 个时间格，每格选择 1 个行动
- 压力、专注、关系、自我认同和任务进度会随行动变化
- 失败后回到早晨，但会保留上一轮获得的笔记和建议
- 好结局要求完成任务，同时维持压力、关系和自我认同的平衡

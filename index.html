                  游戏说明手册

                  背景

                  你是一名学生，今天是课程作品提交前的最后一天。你的时间只有 6 格——早晨、上午、中午、下午、傍晚、夜晚。每次行动消耗一格时间。如果压力爆表或夜晚到来，今天结束。

                  好消息是：失败后可以重启今天。更好的消息是：线索和记忆会保留到下一轮。

                  五种状态

                  • 压力（红色）：达到 100 立即过载失败。赶工、连续赶工、随机事件会增加；休息、聊天、求助可降低。
                  • 专注（蓝色）：会影响学习效率。专注较高时学习额外增加进度，专注过低时学习收益下降。
                  • 关系（绿色）：找朋友聊天可提升。关系达到 50 或获得求助线索后，可以求助。
                  • 自我认同（紫色）：做兴趣活动大幅提升。低于 50 即使交了作品也是"空心完成"。
                  • 任务进度（橙色）：赶工 +25，求助 +30。达到 80 才有好结局资格。

                  今日策略

                  • 稳住节奏：休息额外降低压力，学习压力略微降低，适合避免过载。
                  • 冲刺提交：学习和求助额外推进进度，但学习会带来更多压力。
                  • 先找连接：聊天获得更多关系，求助收益略高，适合快速打开求助路线。

                  五种行动

                  • 学习 / 赶工：基础任务进度 +25，专注 +15，压力 +20。连续选择额外 +15 压力；专注和策略会改变实际收益。
                  • 休息：压力 -25，专注 +5。压力 ≥ 80 时休息会睡过头，跳过下一格时间。
                  • 找朋友聊天：基础关系 +20，压力 -10。中午去天台会触发特殊线索和求助提示。
                  • 做兴趣活动：自我认同 +30，压力 -10。不加进度，但影响结局。
                  • 向老师 / 学长求助：基础任务进度 +30，压力 -10。需要关系足够或获得线索。

                  成功条件

                  不是把每一格都填满任务，而是达到好结局的平衡条件：

                  • 任务进度 ≥ 80
                  • 压力值 < 90
                  • 自我认同 ≥ 50
                  • 至少找过朋友或求助一次

                  五种结局

                  • 过载失败：压力 ≥ 100，时间倒流，记忆保留
                  • 目标未完成：夜晚任务进度 < 80
                  • 空心完成：任务完成但自我认同 < 50
                  • 差一点的一天：任务完成但压力或自我不达标
                  • 不是满分的一天：平衡的胜利

                  五张今天笔记

                  • 连续赶工会让压力失控：不能只靠硬撑
                  • 中午去天台更容易遇到朋友：特定时间特定地点
                  • 关系足够时可以开口求助：积累是关键
                  • 兴趣活动会影响最后的自我认同：做自己很重要
                  • 休息不能拖到完全崩溃之后：预防胜于治疗

                  游戏主题

                  《重启今天》不表达"不要努力"，也不鼓励"躺平"。它表达的是：
                  真正的成长不是无限消耗自己，而是学会取舍、求助、休息，并保留自己的节奏。

                  90 秒演示路线

                  评委快速体验建议

                  这条路线能在短时间内展示“失败 → 便签 → 重启 → 调整 → 好结局”的完整闭环。

                  1. 连续赶工：连续选择“学习 / 赶工”，展示任务进度上升但压力快速累积。
                  2. 过载重启：压力爆表后进入“过载失败”，今天笔记新增便签。
                  3. 发现连接：重启后尝试找朋友聊天，中午天台会解锁求助线索。
                  4. 组合通关：安排学习、朋友、求助、休息和兴趣活动。
                  5. 主题落点：展示好结局“不是满分的一天”。

                  推荐通关路线

                  稳住节奏 → 学习 / 赶工 → 休息 → 找朋友聊天 → 向老师 / 学长求助 → 学习 / 赶工 → 做兴趣活动。
